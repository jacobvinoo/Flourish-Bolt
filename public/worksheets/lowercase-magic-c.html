Worksheet 3.1: The "Magic C" Letters

"All these letters start with the same 'c' shape! c, a, d, g, o, q"

The Magic C Shape

This is the "Magic C" - it starts every letter below!

Letter c (just the Magic C!)

Letter a (Magic C + line down)

Letter d (Magic C + tall line up)

Letter g (Magic C + tail down)

Letter o (Magic C + close the circle)

Letter q (Magic C + line down with tail)

✨ Magic C Tips:

  • • All letters start the same way - with the Magic C shape!
  • • c: Just the Magic C by itself
  • • a: Magic C + straight line down on the right
  • • d: Magic C + tall line up and down on the right
  • • g: Magic C + line down with a hook at the bottom
  • • o: Magic C but close it into a complete circle
  • • q: Magic C + straight line down below the baseline
  • • Start at the top right and curve around to the left!
